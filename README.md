# Book Tracker - Трекер прочитанных книг

**Автор:** Фёдоров Никита Александрович

## Описание
Приложение для учёта прочитанных книг. Позволяет добавлять книги, фильтровать по жанру и количеству страниц, сохранять и загружать данные в JSON.

## Технологии
- Python 3
- Tkinter (встроенный GUI)
- JSON для хранения данных

## Установка и запуск
1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `book_tracker.py`
3. Запустите командой:
```bash
python book_tracker.py
